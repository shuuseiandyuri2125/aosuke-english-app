# 蒼佑くんの英単語アプリ

## スマホで一発で開くために必要なこと

このアプリは、スマホのブラウザで開けるWebアプリです。
商品として使う場合は、このフォルダをNetlify、Vercel、GitHub Pagesなどの静的ホスティングにアップロードし、HTTPSの公開URLを発行してください。

公開URLができれば、スマホでは次の形で使えます。

- URLをタップしてすぐ起動
- QRコードから起動
- ホーム画面に追加してアプリ風に起動
- 一度読み込んだあと、基本ファイルはオフラインでも再利用

## もっとも簡単な公開方法

Netlifyの「Deploy manually」に、この `aosuke-english-app` フォルダをドラッグ&ドロップします。
発行されたURLをスマホで開けば、そのまま使えます。

## 音声について

音声は端末の読み上げ機能を使っています。
iPhone、Android、Chrome、Safariなどの環境差で声質は変わりますが、外部音声APIなしでワンタップ再生できます。

## 単語を増やす場所

単語は `app.js` の `WORDS` に入っています。
次の形で追加できます。

```js
{ grade: 1, unit: "学校・持ち物", word: "book", meaning: "本", image: "BOOK", kind: "もの" }
```

## 商品化前の確認項目

- iPhone Safariで音声ボタンが鳴るか
- Android Chromeで音声ボタンが鳴るか
- 5語/10語の切り替えが自然か
- 蒼佑くん本人が迷わず「音を聞く」「答える」まで進めるか
- 公開URLをQRコード化して、初回アクセスが迷わないか
